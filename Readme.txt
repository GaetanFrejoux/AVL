Projet réalisé par Fréjoux Gaëtan et Niord Mathieu dans le cadre de notre enseignement
en algorithmique et programmation.

Pour tester ce projet, il vous faudra modifier le fichier : abr.ml
afin de modifier le module pour qu'il soit compatible à votre version d'ocaml.

Chaque partie de ce projet est accompagnée d'un fichier d'expérimentations de manière à ce que les fonctions
ne soient pas mêlées pas aux tests.
